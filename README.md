# RuCollection
## Description
Collection of SCP:SL plugins for russian server created by iFoundation community, using Exiled API. 
## Plugins
- Subclasses (Contains SCP-343, SCP-181).
- ScpSwap.
- RemoteKeycard.
- InfinityAmmo.
- Grenades for Chaos.
- FirearmMechanics.
- RealisticBleeding.
- ZombieInfection.
- BetterPainkillers.
- ExpForGenerators.
- SCP-035 (Planned?).
- RoundScenarious (Planned?).
